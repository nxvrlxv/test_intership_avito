# PR Reviewer Assignment Service (Fall 2025)

Сервис для автоматического назначения ревьюверов на Pull Request по командам.

## Особенности реализации

- FastAPI + SQLAlchemy + Pydantic
- Автоматическое назначение 2 активных участников из той же команды (исключая автора)
- Поддержка переназначения ревьювера
- Запуск одной командой docker compose up
- Для наполнения БД использовать файл seed_db.py
